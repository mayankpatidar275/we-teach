## How to Play

- **Jump**: Use the **up arrow key**.
- **Move**: Use the **right and left arrow keys**.
- **Pause**: Press the **spacebar** to pause the game.

## Technologies Used

- HTML
- CSS Animation
- Vanilla JavaScript
